# 量化核心概念指南

> 本文档面向所有参与或使用 Bili_Stock 项目的人，无需量化基础即可理解。

---

## 1. 我们在做什么

我们追踪**雪球组合**（Xueqiu Cubes）中精英投资者的交易行为。当多个聪明钱组合在短时间内买入同一只股票，这就是一个做多信号。简单说：**跟着聪明人的共识方向走**。

核心数据：
- **cubes.db**：55,000+ 个雪球模拟组合，其中 291 个有完整的调仓记录
- **rebalancing_history**：8,070 条成功的调仓记录（买入/卖出/调整仓位）
- **覆盖时间**：2014-2026 年，1,373 只股票

## 2. 核心指标

### 2.1 IC（Information Coefficient，信息系数）

**一句话：IC 衡量"你的预测有多准"。**

每个调仓日，我们把所有股票按信号值排序，然后看排名靠前的股票是不是真的涨得多。

```
IC = corr(信号排名, 未来两周实际收益排名)
```

| IC 值 | 含义 | 类比 |
|-------|------|------|
| +0.05 以上 | 非常强 | 天气预报说晴天，80%的时候真晴 |
| +0.02 ~ +0.05 | 不错 | 天气预报准确率比随机高不少 |
| +0.005 ~ +0.02 | 微弱但有用 | 比抛硬币好一点，但靠量取胜 |
| 0 | 没用 | 纯猜，跟抛硬币一样 |
| 负数 | 反向 | 信号说涨，实际跌 — 要么别用，要么反着用 |

**我们的信号 IC 在 0.002~0.004 之间。** 看起来很小，但：
- 每 12 天调仓一次
- 每次覆盖几百只股票
- 15 年累积下来，这个微小的优势就变成了真金白银

### 2.2 ICIR（IC 的稳定性）

```
ICIR = IC均值 / IC标准差
```

IC 如果今天 +0.05 明天 -0.04，平均虽然正但波动很大，赚的钱可能被回撤吞掉。ICIR 衡量的是**信号稳定地指向正确方向的程度**。

| ICIR | 含义 |
|------|------|
| > 0.5 | 优秀，信号非常稳 |
| 0.2 ~ 0.5 | 不错 |
| < 0.1 | 信号虽然平均为正，但日间波动太大 |

### 2.3 Calmar Ratio（风险收益比）

```
Calmar = 年化收益率 / 最大回撤
```

**这是我们评判策略好坏的首要指标。** 它同时考虑了赚钱能力和控制亏损的能力。

| Calmar | 含义 |
|--------|------|
| < 0.1 | 差 — 赚的少，亏的多 |
| 0.2 ~ 0.5 | 可以接受 |
| 0.5 ~ 1.0 | 不错 |
| > 1.0 | 优秀 |

**例子：**
- calmar = 0.10 → 年赚 2.5%，曾经最多亏 25%（不划算）
- calmar = 0.48 → 年赚 6.9%，最多亏 14.4%（Phase 2 优化后的成绩）

### 2.4 MDD（Maximum Drawdown，最大回撤）

从历史最高点到最低点的最大跌幅。**衡量"最坏情况下你要亏多少"。**

```
MDD = -14.4%  →  如果在最高点投入 100 万，最多跌到 85.6 万
```

### 2.5 Sharpe Ratio（夏普比率）

```
Sharpe = (年化收益 - 无风险利率) / 年化波动率
```

和 Calmar 类似，但用波动率替代最大回撤来衡量风险。Sharpe > 0.5 算不错，> 1.0 算优秀。

## 3. 市况分类（Regime）

我们把 A 股市场按沪深300指数的 20 日涨跌幅分为三种状态：

| 市况 | 判定条件 | 占比 | 信号表现 |
|------|----------|------|----------|
| **上涨（牛市）** | 20日涨幅 > 2% | ~35% | IC ≈ +0.001（微弱） |
| **震荡（横盘）** | 涨幅在 -2%~+2% | ~31% | IC ≈ -0.001（噪声） |
| **下跌（熊市）** | 20日跌幅 > 2% | ~34% | IC ≈ +0.010（最强） |

**关键发现：** 信号在熊市最好用（聪明钱抄底准确），在震荡期基本是噪声。这就是为什么我们在震荡期缩减仓位 — 既然信号没用，不如少亏点。

## 4. 我们的信号是怎么构建的

### 4.1 原始信号：net_buy_cube_count（计数信号）

```
每天，对每只股票：
  数有多少个不同的雪球组合在买入这只股票
  → net_buy_cube_count = 3 表示今天有 3 个组合买了

然后做两步处理：
  1. 和 14 天前的值比较（动量）
  2. 3 天滑动平均（平滑噪声）
  3. 跨股票标准化（z-score）
```

**问题：** 只数人头，不看仓位大小，完全忽略卖出。

### 4.2 新信号：net_flow（净流量信号）

```
每天，对每只股票：
  净流量 = 所有买入的仓位变化之和 - 所有卖出的仓位变化之和

例子：
  组合 A 买入 +5% 仓位
  组合 B 买入 +2% 仓位
  组合 C 卖出 -8% 仓位
  → net_flow = 5 + 2 - 8 = -1（净卖出，看空信号）
```

**net_flow 的优势：** 在震荡期 IC 从 -0.001 翻转到 +0.005（正的！），因为聪明钱的**卖出信号**在震荡市比买入更有信息量。

### 4.3 信号处理流水线

```
cubes.db 原始数据
  → 信号计算（count 或 net_flow）
    → 行业中性化（去掉行业偏差）
      → 流动性过滤（剔除冷门股）
        → 市况分类（上涨/震荡/下跌）
          → 选股 + 风控
            → 回测 → Calmar / Sharpe / MDD
```

## 5. 关键发现（按时间顺序）

### Phase 1：投产基准（choppy_fix_B）
- 建立了生产基线，Calmar = 0.28（2019-2025）
- 实盘验证胜率 53.76%

### Phase 2：震荡优化
- **核心发现：震荡期 IC = -0.001，占 31% 交易日 = 在噪声上交易**
- 解决方案：在震荡期亏损日缩减仓位到 30%（保留盈利日全仓位）
- **结果：Calmar 0.10 → 0.48（+557%），MDD 从 -25% 降到 -14%**
- 也测试了"震荡期完全空仓"，反而更差（丢弃了盈利的震荡期）

### Phase 3：信号升级（进行中）
- 发现 net_flow（买-卖净流量）在震荡期 IC = +0.005
- **如果成立：不需要躲避震荡，可以全天交易，等于多利用 31% 的交易日**
- 回测正在运行中

## 7. 常见术语对照

| 术语 | 全称 | 含义 |
|------|------|------|
| IC | Information Coefficient | 因子预测收益的相关系数 |
| ICIR | IC Information Ratio | IC 的稳定性（均值/标准差） |
| Calmar | Calmar Ratio | 年化收益 / 最大回撤 |
| MDD | Maximum Drawdown | 历史最大跌幅 |
| Sharpe | Sharpe Ratio | 风险调整后收益 |
| factor_z | Factor Z-score | 标准化后的因子值 |
| factor_z_neu | Neutralized Factor | 行业中性化后的因子值 |
| regime | Market Regime | 市况分类（上涨/震荡/下跌） |
| go-flat | Go Flat | 空仓，不持有任何头寸 |
| hold_step | Holding Period | 持仓周期（每 N 个交易日调仓一次） |
| SRF | SmartResonanceFactor | 多因子综合评分（已测试，效果不好） |
| net_flow | Net Flow | 买卖净流量 |
| fwd_ret_2w | Forward Return 2 Weeks | 未来两周实际收益率 |
